Real-Time Shelter Resource Monitor

A live capacity tracking system that streamlines resource allocation and verifies real-time availability for emergency shelters.

  • Python (PyQt5/Tkinter) front-end + PostgreSQL backend
  • Cloud synchronization across multiple shelters
  • Built for operational speed and staff usability

Impact: Reduces delays and improves emergency housing access by providing transparent, real-time availability.
